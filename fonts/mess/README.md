# Mess

*Mess* is a typeface designed by [Tezzo Suzuki](http://tezzosuzuki.com/) and released by [Velvetyne Type Foundry](http://velvetyne.fr/fonts/mess/).

Font is something like a corporation. From outside, it is hard for us to see how it works and who is working in it. In metal type era, we were close to glyphs but today we never know how they feels about their treatment.
